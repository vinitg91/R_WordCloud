# R_WordCloud
Requires following libraries:

library(shiny)
library(devtools)
library(wordcloud2)
library(tm)
library(SnowballC)
library(wordcloud)
library(readxl)
library(xlsx)

Users can launch app by running the following command in R console:
runGitHub( "R_WordCloud", "vinitg91") 

Or visit https://vinitg.shinyapps.io/r_wordcloud/
